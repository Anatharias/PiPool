# PiPool
## Requirements

### Python and the following modules
```
pip install RPi.GPIO smbus2 w1thermsensor gpiozero bh1750 raspberrypi-tm1637
```

### for Thingsboard:
```
pip install tb-mqtt-client
```

- Temperature sensors : BH18B20
- Sun sensor : BH1750
- Simple relay
- TM1737 displays
